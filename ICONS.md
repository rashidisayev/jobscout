# Icon Assets

The extension requires three icon files:
- `assets/icon16.png` (16x16 pixels)
- `assets/icon48.png` (48x48 pixels)  
- `assets/icon128.png` (128x128 pixels)

## Creating Icons

You can create these icons using:

1. **Online Tools**:
   - [Favicon Generator](https://www.favicon-generator.org/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)

2. **Image Editors**:
   - Use any image editor (Photoshop, GIMP, etc.) to create 16x16, 48x48, and 128x128 pixel images
   - Recommended color: #0077b5 (LinkedIn blue) or any color of your choice
   - Simple design: A magnifying glass, briefcase, or "JS" text works well

3. **Quick Placeholder**:
   - For testing, you can use any PNG images with the correct dimensions
   - The extension will work with placeholder icons

## Recommended Design

A simple design suggestion:
- Background: #0077b5 (LinkedIn blue) or transparent
- Icon: White magnifying glass or briefcase symbol
- Text: "JS" for JobScout

Save the same design at three different sizes (16, 48, 128 pixels) as PNG files.
